Solicitar un Conjunto | BETA

Datos.gob.mx incluirá mecanismos para la solicitud ciudadana de Datos Abiertos, las cuales seguirán el siguiente procedimiento:

  1. Las solicitudes serán remitidas al correo electrónico del Administrador de Datos
  2. El Administrador de Datos deberá consultar a las Áreas Generadoras sobre la disponibilidad de los Datos Solicitados y si la consulta procede o no
  3. De tener una respuesta afirmativa, el Administrador de Datos dará respuesta a la solicitud especificando la fecha estimada de publicación de los datos en datos.gob.mx
  4. De tener respuesta negativa, el Administrador de Datos dará respuesta a la solicitud con el fundamento para no publicar los datos

Las instituciones tendrán la obligación de responder a las solicitudes en un periodo de 40 días naturales a partir de que se informen al Administrador de Datos.

Todas las solicitudes de datos ciudadanas, su estatus dentro de las instituciones y las respuestas a las mismas serán públicadas en datos.gob.mx
